O que é um Hackaton?

Hackathon significa maratona de programação. O termo resulta da junçao das palavras inglesas “hack” (programar de forma excepcional) e “marathon” (maratona). Ou seja, o Hackathon é um evento que reúne programadores, designers e outros profissionais ligados ao desenvolvimento de software para uma maratona de programação, cujo objetivo é desenvolver um software ou solução tecnológica que atenda a um fim específico, onde normalmente dura alguns dias, para que ao final, a equipe tenha desenvolvido uma soluçao para o problema.
